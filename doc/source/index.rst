########################
libfacerec documentation
########################

`libfacerec <http://www.github.com/bytefish/libfacerec>`_  is a face 
recognition library for the `OpenCV2 <http://www.opencv.org>`_ C++ API. 
It provides implementations of the Eigenfaces [TP91]_ method, Fisherfaces method
[Belhumeur97]_ and Local Binary Patterns Histograms [Ahonen04]_.

This document describes the libfacerec :doc:`API </api>`. It comes with 
:doc:`tutorials </tutorial>` to (1) show the usage idioms, (2) see how the 
algorithms perform on real life data and (3) explain the math behind the
algorithms.
